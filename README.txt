1. This time "L4_HARQ.c" needs to be modified, as indicated during the class.

2. Please make sure you have the latest version of "gcc"

3. Compile with command line:
	make

4. Execute with command line.
	./L4_HARQ_ofdm 8 20000
   E.g.: 8 means taps - the length of the channel delay profile
         20000 means symbol blocks - the number of symbol blocks to simulate
   Your results will be stored in a .txt file in 'results' folder.

5. Plot your graphs with any tools you like (e.g., Python, ...)
